## 项目的初始化
- cli工具安装
```bash
npm install -g @tarojs/cli
或者使用 yarn 方式
yarn global add @tarojs/cli
```
- 使用taro命令初始化项目
```bash
taro init newApp
```
-  初始化完成后,安装相关依赖
```bash
npm install 
or 
npm i
```
- 运行项目

需要在微信开发者平台，注册自己的小程序，拿到appid, 使用微信开发者工具打开自己的项目。运行一下命令，可以看到相应的UI页面了
  ```bash
  npm run dev:weapp
  ```

- 完成以上步骤，项目初始化已经完成
